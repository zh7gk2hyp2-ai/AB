# تفعيل التحليلات (صديقة للخصوصية · بدون كوكيز)

الموقع مهيّأ لاستخدام **Cloudflare Web Analytics** — مجانية، بدون كوكيز، ولا تجمع بيانات شخصية عن الزوّار.

## الخطوات (دقيقتان)
1. ادخل على Cloudflare Dashboard → **Analytics & Logs** → **Web Analytics**.
2. اضغط **Add a site** وأدخل رابط موقعك.
3. انسخ قيمة **token** الظاهرة في الكود (JS snippet).
4. افتح الملف `assets/analytics.js` وضع التوكن بين علامتي التنصيص في السطر:
   ```js
   var TOKEN = "ضع_التوكن_هنا";
   ```
5. احفظ وارفع التغيير. تبدأ الإحصاءات بالظهور خلال دقائق.

> ملاحظة: ما دام `TOKEN` فارغاً، لا يُحمّل أي سكربت خارجي إطلاقاً — الموقع يبقى بلا تتبّع.

## بدائل صديقة للخصوصية (اختياري)
- **GoatCounter** (مجاني، مفتوح المصدر): استبدل محتوى `assets/analytics.js` بكود GoatCounter.
- **Plausible / Umami**: تحتاج اشتراك أو استضافة ذاتية.
